# モジホコリ — 展示概要

## 概要

「モジホコリ」は、筑波大学の授業「メディアアート」のために制作している、変形菌（Physarum）に着想を得たインタラクティブ作品です。

画面上では、黄色い生命体が周囲にある餌を探し、自らが残した経路を強化しながら網目状の形を作ります。餌や障害物の位置が変わると、それまでの形を少しずつ組み替え、新しい環境に適応していきます。

この作品では、あらかじめ決められた映像を鑑賞するのではなく、鑑賞者が環境に手を加え、その結果として生まれる予測しきれない形や動きを観察する体験を目指しています。

## 作品の体験

最終的な展示では、ディスプレイを水平に設置し、その上に次のような実物のオブジェクトを置く構成を想定しています。

- キノコのような形をした「餌」
- 石のような形をした「障害物」

鑑賞者はこれらのオブジェクトを手で動かします。作品はその位置を画面内の餌や障害物として反映し、変形菌を思わせる生命体が、新しい配置に応じて進路や形を変えていきます。

鑑賞者の主な行為は、何かを直接操作して正解を目指すことではありません。オブジェクトを置く、少し待つ、変化を観察する、再び配置を変える、という往復を通して、環境と生命の関係を眺める作品です。

## 現在のプロトタイプ

現在は、ブラウザ上で動作するシミュレーションとして実装しています。p5.jsを使用し、多数の粒子が餌からの誘引と自らの軌跡を手がかりに移動することで、有機的な形を生成します。

現時点では、次の操作が可能です。

- 画面上の餌と障害物をドラッグして移動する
- 「朽ち木」「腐葉土」「湿った岩」の3種類から背景環境を切り替える
- 成長速度を変更する
- 全画面で表示する
- シミュレーションを再生成、一時停止、画像として保存する

餌には栄養量があり、生命体に覆われると徐々に消費されます。また、よく通る経路は強化され、使われない経路は薄れていくため、時間の経過とともに形が変化し続けます。

## 最終展示のイメージ

水平に置いたディスプレイを箱の中に収め、その周囲を土などの自然素材で覆うことを考えています。機器としてのディスプレイをそのまま見せるのではなく、土の中に発光する観察面が埋め込まれているような外観にすることで、画面内の生命体と実物のオブジェクトが同じ環境に存在しているように感じられる展示を目指します。

想定している構成要素は次のとおりです。

- 水平に設置するディスプレイ
- シミュレーションを動かすコンピューター
- 餌と障害物を模した実物のオブジェクト
- オブジェクトの位置を検出し、シミュレーションへ反映する仕組み
- ディスプレイを収める箱と、その周囲を構成する土などの素材

## 今後の検討事項

- 実物の餌と障害物をどのような方法で識別し、位置を取得するか
- 実物の位置と画面内の座標をどのように対応させるか
- 展示中に安定して連続稼働させるための起動、復旧、キャリブレーション方法
- 土や造形物を使用しながら、ディスプレイの放熱、保護、安全性を確保する方法
- 鑑賞者が説明を読まなくても、オブジェクトを動かせることに気づける見せ方

本リポジトリでは、まず画面上での生命体の振る舞いとインタラクションを制作し、今後、実物のオブジェクトを使った展示システムへ発展させていきます。
